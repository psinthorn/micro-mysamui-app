# Shadcn Install Command

```bash

# Install Shadcn
npx shadcn@latest init

# Install Shadcn button
npx shadcn@latest add button

# Install Shadcn avatar
npx shadcn@latest add avatar

# Install Shadcn dropdown menu
npx shadcn@latest add dropdown-menu

# Install Shadcn command
npx shadcn@latest add command

```

```html
DropdownMenuSeparator, DropdownMenuTrigger, } from
"@/components/ui/dropdown-menu" # Add Shadcn Dropdown Menu
<DropdownMenu>
  <DropdownMenuTrigger>Open</DropdownMenuTrigger>
  <DropdownMenuContent>
    <DropdownMenuLabel>My Account</DropdownMenuLabel>
    <DropdownMenuSeparator />
    <DropdownMenuItem>Profile</DropdownMenuItem>
    <DropdownMenuItem>Billing</DropdownMenuItem>
    <DropdownMenuItem>Team</DropdownMenuItem>
    <DropdownMenuItem>Subscription</DropdownMenuItem>
  </DropdownMenuContent>
</DropdownMenu>
```
